# Contributing to TRMNL Screenshot Integration

Thank you for your interest in contributing!

## How to Contribute

### Reporting Issues
- Check existing issues first
- Include error logs and steps to reproduce
- Specify your Home Assistant version

### Submitting Changes

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test thoroughly
5. Submit a pull request

## Code Style

### Python
- Follow PEP 8
- Use type hints
- Add docstrings
- Keep it Pythonic

### Testing
- Test the full capture-and-send workflow
- Test error handling and edge cases
- Verify integration with both Phase 1 and Phase 2

## Development Setup

```bash
# Clone and copy to Home Assistant
git clone <your-fork>
cp -r custom_components/trmnl_screenshot ~/.homeassistant/custom_components/
```

## Commit Guidelines

- Use clear, descriptive messages
- Reference issues when relevant
- One feature per commit when possible
